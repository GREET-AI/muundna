# Landing-Builder – ausschließlich für Builder / Produkt-Landingpages

**Diese Komponenten sind exklusiv für den Landingpage-Builder und die Produkt-Seiten unter `/p/[slug]` sowie die Admin-Vorschau.**

- Änderungen am Builder oder an Produkt-Landingpages **nur hier** vornehmen.
- Die **Hauptwebsite** (Startseite, Handwerker/Dienstleistungen) nutzt die Komponenten in `app/components/` direkt und darf von Änderungen hier **nicht** betroffen werden.
- Keine Imports aus diesem Ordner in `app/page.tsx` oder anderen reinen Website-Seiten.
